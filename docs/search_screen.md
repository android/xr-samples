# Search screen
XR 환경 상에서 인스타 탐색창을 만든다.

하단에 별도로 위치한 네비게이션 바의 돋보기 버튼을 터치하면 추천 게시물 화면으로 들어가게 된다.

정사각형 또는 9:16비율의 릴스가 섞여서 나오고, 상하좌우 어디로 드래그 하든, 가장자리에 새로운 추천 게시물이 보이게 된다.

특정 게시물을 핀치하거나 컨트롤러를 이용하여 터치하면 해당 게시물이 별도의 창으로 눈앞에 나타나고, 거기서 아래로 스크롤 하면 기존의 인스타와 동일하게 다른 추천 게시물이 계속 보이게 된다.

게시물 데이터는 일단 별도의 데이터베이스를 연결하지 않고, 샘플 데이터들을 하드코딩하여 이용한다.
